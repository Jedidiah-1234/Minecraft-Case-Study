Critics

Minecraft received widespread acclaim from critics. Reviewers praised its creative gameplay, freedom, and replayability. IGN gave it a score of 9/10, while PC Gamer called it a "landmark in gaming history". Its simple yet deep mechanics allowed players to build, explore, and survive in ways that set new standards for sandbox games.

Sales

As of 2024, Minecraft has sold over 300 million copies, making it the best-selling video game of all time. It has a massive player base across platforms including PC, consoles, and mobile. Its success has driven continuous updates and spinoffs, such as Minecraft Dungeons and Minecraft Legends.

Awards

Minecraft has won multiple awards including:

  • Golden Joystick Award for Best Downloadable Game (2012)
  • Kids’ Choice Award for Favorite Video Game (multiple years)
  • BAFTA Special Award for Game Innovation

It was also inducted into the World Video Game Hall of Fame in 2020.

2014 EULA Changes

In 2014, Mojang updated Minecraft’s End User License Agreement (EULA), restricting monetization on multiplayer servers. This caused backlash from server owners who relied on donations. However, Mojang clarified that cosmetic or non-pay-to-win monetization would still be allowed.

Account Migration

In 2021, Mojang began requiring players to migrate from Mojang accounts to Microsoft accounts. This move aimed to improve security but received criticism for being mandatory and causing issues for some players during the transition.

Java Edition Chat Reporting

In 2022, Minecraft added a chat reporting feature to Java Edition, allowing players to be reported for inappropriate behavior. This sparked major backlash in the community, with some fearing that it would lead to false bans or censorship. Mojang maintained that it was to ensure player safety.

Mob Vote

Each year, Minecraft players vote for new mobs to be added to the game. This Mob Vote has stirred controversy, especially when popular choices like the Copper Golem lost. Some fans argue it causes division, and others believe all proposed mobs should be added eventually.
