Development

Java Edition

The original version of Minecraft for Windows, macOS, and Linux. Known for modding support and advanced features like snapshots and custom servers.

Pocket Edition

Initially released for mobile platforms. Offered basic Minecraft gameplay with touchscreen controls. Later merged into Bedrock Edition.

Bedrock and Console Editions

A unified version across mobile, console, and Windows 10/11. Supports cross-play, Realms, and Marketplace content.

Other Versions

Minecraft Education Edition

Used in classrooms to teach coding, history, science, and collaboration. Includes educator tools and lesson templates.

China Edition

A free-to-play version of Minecraft made for the Chinese market. Managed by NetEase, it includes local features and restrictions.

Minecraft for Windows

The Bedrock-based version for Windows 10 and 11 with better performance, ray tracing support, and Xbox integration.

Game Variants

Includes versions like Minecraft Dungeons, Earth (discontinued), and other experimental or themed adaptations of the core game.

Virtual Reality

Minecraft VR lets players experience the game in a fully immersive 3D environment using devices like Oculus Rift and HTC Vive.
